### 🎯 使用方法

#### 创建私聊：
1. 点击侧边栏的联系人按钮（👥）
2. 在联系人列表中**双击**任意团队成员
3. 系统自动创建私聊并跳转到聊天页面

#### 创建群聊：
1. 点击侧边栏的联系人按钮（👥）
2. 点击"创建群聊"按钮
3. 输入群聊名称
4. 选择 2 个及以上成员（点击选中/取消）
5. 点击"确认创建"按钮
6. 系统自动创建群聊并跳转到聊天页面

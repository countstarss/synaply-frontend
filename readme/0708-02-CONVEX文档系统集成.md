## 已创建的 Convex 文档系统组件

### 1. **ConvexDocsContext.tsx** - 核心上下文
- 提供完整的文档管理功能
- 支持文档和文件夹的创建、更新、删除
- 集成 Convex mutations 和 queries
- 管理打开的文档状态和标签页

### 2. **ConvexDocsSidebar.tsx** - 文档侧边栏
- 支持文件夹和文档的层级结构显示
- 实现右键菜单功能（创建、重命名、删除）
- 支持拖拽展开/收起文件夹
- 实时更新文档树结构

### 3. **ConvexBlockNoteEditor.tsx** - 编辑器组件
- 适配 Convex 的 BlockNote 编辑器
- 支持实时保存和自动保存
- 键盘快捷键支持 (Cmd/Ctrl + S)
- 主题自适应

### 4. **ConvexFolderIntro.tsx** - 文件夹页面
- 文件夹描述编辑功能
- 权限显示和编辑控制
- 使用提示和指导

### 5. **ConvexDocsEditor.tsx** - 主编辑器
- 整合 BlockNote 编辑器和文件夹介绍页面
- 支持标题编辑
- 权限控制和状态显示
- 展开/收起工具栏

### 6. **ConvexDocsTabs.tsx** - 标签页管理
- 显示打开的文档和文件夹
- 支持关闭标签页
- 区分文档和文件夹图标
- 响应式布局

### 7. **ConvexDocsPage.tsx** - 主页面
- 整合所有组件
- 支持展开/收起侧边栏
- 加载状态处理
- 欢迎页面和空状态

## 主要特性

✅ **完整的文档系统** - 支持文档创建、编辑、组织  
✅ **文件夹层级结构** - 支持无限层级的文件夹嵌套  
✅ **实时同步** - 基于 Convex 的实时数据同步  
✅ **权限控制** - 支持私有、团队只读、团队可编辑等权限  
✅ **现代化编辑器** - 基于 BlockNote 的富文本编辑  
✅ **响应式设计** - 适配不同屏幕尺寸  
✅ **键盘快捷键** - 提高操作效率  
✅ **自动保存** - 防止数据丢失
